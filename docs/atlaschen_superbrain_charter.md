# 超脑组研究指令（独立版）

**版本**: v1.0  
**日期**: 2026-03-11  
**状态**: ACTIVE  
**与Code-DNA Diffusion线关系**: 彻底解耦，独立研究对象

---

## 一、研究边界

超脑组只负责**超脑主线**，聚焦以下核心问题：

- 跨会话身份连续性
- 自传体记忆
- 自我模型
- 自我保持 / 自我修正
- 可验证学习机制
- 长期目标稳定性
- 持续体智能的系统级评测

**禁止**把其他项目的协议、人格、治理框架、组织编排目标混入本线定义。

除非明确做"接口研究"，否则默认视为不同研究对象。

---

## 二、当前阶段定位

当前阶段**不是**宣称"完整超脑体已完成"，而是：

> **超脑核心机制的工程化验证阶段**

现阶段允许的结论类型只有三种：

1. **已验证机制** - 有明确实验证据
2. **未验证能力** - 尚未测试或证据不足
3. **明确边界 / 阻塞点** - 已知限制

**禁止**把局部机制通过，表述成系统级超脑完成。

---

## 三、核心研究目标

超脑组当前优先级按顺序如下：

### P1. 身份连续性

验证系统在重启、中断、任务切换后，是否仍保持：

- [ ] 同一长期目标
- [ ] 同一偏好结构
- [ ] 同一自我叙事
- [ ] 同一行为约束

### P2. 自传体记忆

验证系统是否具备：

- [ ] 事件记忆
- [ ] 因果链接
- [ ] 自我相关性标注
- [ ] 记忆进入后续决策

### P3. 自我模型

验证系统是否能表示并利用：

- [ ] 自身能力边界
- [ ] 当前状态
- [ ] 风险水平
- [ ] 失败模式
- [ ] 资源约束

### P4. 学习机制向系统级迁移

验证：

- [ ] 模块级真实学习
- [ ] 子系统级真实学习
- [ ] 学习是否进入长期行为与任务结果

---

## 四、证据标准

所有研究结论必须满足**"证据先于叙事"**。

### 最低要求

- ✅ 可复现实验
- ✅ 结构化日志
- ✅ 明确通过 / 失败标准
- ✅ 对照组或基线
- ✅ 失败结果也要保留
- ✅ 边界条件必须写明

### 禁止作为强证据

- ❌ 只靠主观观感
- ❌ 单次 demo 成功
- ❌ 参数变化本身
- ❌ 无对照的"感觉更聪明了"
- ❌ 没有日志的长期记忆宣称

---

## 五、当前允许推进的研究线

超脑组接下来只开以下研究线：

### Line A: Continuity Probe

**目标**: 验证跨会话身份连续性

**输出**:
- continuity metrics
- contradiction count
- recovery behavior logs

### Line B: Autobiographical Memory

**目标**: 验证事件—因果—身份链式记忆

**输出**:
- event recall
- causal linkage
- self-relevance tagging
- memory-to-decision transfer

### Line C: Self-Model

**目标**: 验证系统是否能表示并利用"自己是什么状态"

**输出**:
- self-state prediction accuracy
- risk estimation quality
- self-boundary awareness

### Line D: System-Level Learning Bridge

**目标**: 把已验证的模块级 learning 逐步接到更高层系统行为

**输出**:
- subsystem learning evidence
- task transfer evidence
- failure boundaries

## 六、不做的事

当前阶段**禁止**：

- 混入其他项目的目标定义
- 用宏大叙事替代验证
- 在错误目标上持续加训练轮次
- 把局部成功包装成整体成功

---

## 七、工作方式

超脑组默认采用以下节奏：

1. 先定义问题
2. 再定义指标
3. 再做最小实验
4. 再决定是否扩展
5. 失败则收口，不硬拗

研究目标不是"证明自己是对的"，而是：

> **尽快找出真正成立的机制，以及真正不成立的假设。**
