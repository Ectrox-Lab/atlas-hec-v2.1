## 二、明晨分析优先级 (P0→P2)

### P0: Ordered vs Random分布分析
**目标**: 确认不是均值差，而是双峰/双团簇结构

**分析步骤**:
1. 提取CRIT组所有r_final值
2. 按initial_ordered分组
3. 画分布图 (histogram/KDE)
4. **关键判断**:
   - [ ] 是否呈现双峰或明显分离的两个团簇？
   - [ ] 两个分布的重叠度如何？
   - [ ] 是否有 case 落在中间区 (0.4-0.6) 长期徘徊？

**产出**: 分布图 + 分离度量化 (如KL散度)

### P0: 双稳态案例深度分析
**目标**: 找出18/30双稳态案例的共同条件

**分析步骤**:
1. 筛选r_final < 0.3 或 r_final > 0.7的CRIT案例
2. 交叉分析:
   - [ ] 是否集中在某个σ层？
   - [ ] 是否和特定K值相关？
   - [ ] 是否和N相关？
   - [ ] 初始条件的具体影响？

**产出**: 双稳态条件热力图

### P0: 滞后区宽度估算
**目标**: 量化历史依赖的参数区间

**分析步骤**:
1. 对每个(σ, K)组合，计算ordered vs random的差异
2. 找出差异显著的参数区
3. **关键判断**:
   - [ ] 滞后区是否集中在K_c附近？
   - [ ] 滞后区宽度是否随σ变化？
   - [ ] 是否存在"完全滞后"(gap > 0.5)的参数点？

**产出**: 滞后区宽度 vs 参数图

### P1: 轨迹形状分析
**目标**: 比较三组的收敛行为

**分析步骤**:
1. 对比CTRL/CRIT/HIGH的:
   - r_early (0-2000 gens)
   - r_mid (2000-5000 gens)
   - r_late (5000-10000 gens)
2. **关键判断**:
   - [ ] CRIT组是否收敛更慢？
   - [ ] 是否更容易卡在中间态？
   - [ ] 有无"假性收敛"后重新发散的案例？

**产出**: 轨迹演化对比图

### P1: 稳定性深度分析
**目标**: 量化"临界区敏感但不崩"

**分析步骤**:
1. 计算每个case的stability (r variance last 1000 gens)
2. 分组比较:
   - CTRL stability分布
   - CRIT stability分布
   - HIGH stability分布
3. **关键判断**:
   - [ ] CRIT stability是否系统性地高于两端？
   - [ ] 高stability是否和r_final中间值相关？

**产出**: 稳定性分布对比

### P2: 跨σ一致性检验
**目标**: 验证现象是否在不同频率分布下都成立

**分析步骤**:
1. 分别分析σ=0.1, 0.5, 1.0三组
2. **关键判断**:
   - [ ] 滞后效应是否在所有σ层都存在？
   - [ ] 双稳态比例是否有σ依赖性？
   - [ ] K_c位置是否符合预期？

**产出**: 跨σ一致性报告

## 三、升级决策检查点
