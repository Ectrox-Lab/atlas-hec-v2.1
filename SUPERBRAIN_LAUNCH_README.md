## Temporal Architecture

| Module | Clock Speed | Cycle Time | Purpose |
|--------|-------------|------------|---------|
| Mainline | Slow | 6h checkpoint, 24h audit | Reality judgment, policy validation |
| Fast Genesis | Fast | Minutes-hours (event-driven) | Discovery, exploration |
| Bridge | Medium | Continuous rolling | Funnel, audit, queue |
| Akashic | Medium | 1h synthesis | Knowledge inheritance |

**Time Compression**: "Reality 1 second = Code 100 years"
- Event-driven simulation skips stable epochs
- Surrogate filtering eliminates 95-99% of candidates before full sim
- Lineage compression merges similar branches
